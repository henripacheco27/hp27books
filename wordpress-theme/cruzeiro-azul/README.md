# Tema WordPress: Cruzeiro Azul

Tema completo com páginas padrão do WordPress e estética azul inspirada no Cruzeiro.

## Templates incluídos

- `front-page.php`
- `home.php`
- `index.php`
- `single.php`
- `page.php`
- `archive.php`
- `search.php`
- `404.php`
- `comments.php`
- `sidebar.php`
- `searchform.php`
- `template-parts/content.php`
- `template-parts/content-page.php`
- `template-parts/content-none.php`

## Como usar

1. Copie a pasta `cruzeiro-azul` para `wp-content/themes/`.
2. No painel WordPress, vá em **Aparência > Temas**.
3. Ative **Cruzeiro Azul**.
4. Configure menus em **Aparência > Menus** para `Menu Principal` e `Menu Rodapé`.
5. Configure widgets em **Aparência > Widgets** na `Sidebar Principal`.

## Instalação via ZIP

Se preferir instalar direto pelo painel do WordPress via **Enviar tema**:

1. Gere o arquivo ZIP com:
   ```bash
   ./wordpress-theme/package-theme.sh
   ```
2. O arquivo será criado em:
   - `wordpress-theme/dist/cruzeiro-azul.zip`
3. No WordPress, vá em **Aparência > Temas > Adicionar novo > Enviar tema** e envie esse ZIP.
